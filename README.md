# CPU-Scheduling-Algorithms-analysis
My Operating Systems course project which includes simulations and comparisons of different CPU scheduling algorithms like FSFC, SJF, RR, LIF, and a custom LISC

# How to run the program
To run the program, compile it in the linux terminal by running the commands:
gcc CPUScheduler -o cpu
chmod +x cpu
./cpu xx yy OR 
./cpu xx yy >> result.txt (To save the output in the result.txt file)
WHERE xx is the number of processes you want to simulate and yy are the processes that have I/O burst
